# hacklanta-stocks-project
hacklanta - stock analysis program by THEUCHIHA.

Overview
This stack is designed for beginners who want to build a simple stock analysis platform with room to grow into AI-powered features. It focuses on ease of use, minimal setup, and widely supported tools while still allowing you to scale into more advanced analytics later.

Features<br/>
View stock prices (current + historical)<br/>
Basic charts (line graphs, candlesticks)<br/>
Simple indicators:<br/>
- Moving Average<br/>
- RSI (Relative Strength Index)<br/>
Search stocks by ticker (e.g., AAPL, TSLA)<br/>
Basic “buy/sell signal” logic (rule-based)<br/>
<br/>
Backend<br/>
Python + FastAPI<br/>
- Very beginner-friendly<br/>
- Great for AI and data processing<br/>
- Fast and easy to build APIs<br/>
<br/>
AI / Data Processing<br/>
Pandas<br/>
- Data manipulation and analysis<br/>
NumPy<br/>
- Basic numerical operations<br/>
Scikit-learn<br/>
- Simple machine learning models<br/>
<br/>
Data Sources<br/>
SQLite<br/>
- No setup required<br/>
- Good for small projects
