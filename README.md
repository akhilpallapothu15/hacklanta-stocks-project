# hacklanta-stocks-project
hacklanta - stock analysis program by (insert team name).

Overview
This stack is designed for beginners who want to build a simple stock analysis platform with room to grow into AI-powered features. It focuses on ease of use, minimal setup, and widely supported tools while still allowing you to scale into more advanced analytics later.

Features
View stock prices (current + historical)
Basic charts (line graphs, candlesticks)
Simple indicators:
  Moving Average
  RSI (Relative Strength Index)
Search stocks by ticker (e.g., AAPL, TSLA)
Basic “buy/sell signal” logic (rule-based)

Backend
Python + FastAPI
  Very beginner-friendly
  Great for AI and data processing
  Fast and easy to build APIs

AI / Data Processing
Pandas
  Data manipulation and analysis
NumPy
  Basic numerical operations
Scikit-learn
  Simple machine learning models

Data Sources
SQLite
  No setup required
  Good for small projects
